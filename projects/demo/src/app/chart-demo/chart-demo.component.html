Usos y configuraciones de la directiva libChart

Esta página presenta diversos ejemplos de uso de la directiva libChart de la librería ngx-data-visualizer.

1. Gráfico de Columnas Básico

Este ejemplo demuestra una configuración básica de gráfico de columnas que muestra la distribución de estudiantes por año y condición. El gráfico incluye leyendas en la parte inferior, colores personalizados para cada serie y tooltips informativos. Esta configuración es ideal para visualizar datos categóricos a lo largo del tiempo, permitiendo comparaciones claras entre diferentes grupos.

TypeScript
HTML

2. Gráfico de Líneas con Series Múltiples

Esta configuración presenta un gráfico de líneas que muestra la evolución temporal de diferentes series de datos. El tooltip compartido permite comparar valores de múltiples series en un mismo punto temporal, mientras que las leyendas posicionadas a la derecha facilitan la identificación de cada línea. Este tipo de visualización es especialmente útil para análisis de tendencias y patrones temporales en datos continuos.

TypeScript
HTML

3. Gráfico de Columnas Apiladas

Este ejemplo avanzado muestra un gráfico de columnas apiladas que permite visualizar la composición total de cada categoría. La configuración de apilamiento ("stacked") agrupa las series relacionadas, mientras que el tooltip compartido muestra tanto los valores individuales como el total acumulado. Esta visualización es perfecta para mostrar partes de un todo y cómo cambian las proporciones a lo largo del tiempo.

TypeScript
HTML

4. Gráfico de Columnas con eje x doble nivel

Este ejemplo avanzado muestra un gráfico de columnas con doble nivel en el eje x, y permite visualizar el año y el sector.

TypeScript
HTML

5. Funcionalidades Interactivas

Esta sección demuestra las capacidades interactivas de la directiva libChart. Incluye métodos para alternar entre vista normal y porcentual, así como opciones de exportación en formatos SVG y JPG. Los botones interactivos permiten cambiar dinámicamente el modo de visualización y descargar el gráfico en diferentes formatos, proporcionando flexibilidad para presentaciones y reportes.
