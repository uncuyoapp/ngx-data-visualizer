Guía de Configuración de ngx-data-visualizer

1. Importación de la Librería

Para comenzar a usar ngx-data-visualizer, debes importar los módulos y directivas necesarios en tu componente o módulo de Angular.

TypeScript

2. Directivas Disponibles

La librería proporciona las siguientes directivas que debes importar y utilizar según tus necesidades:

  • ChartDirective: Para visualización de datos en gráficos individuales. Se utiliza con el selector libChart.
  • MultipleChartDirective: Para visualización de múltiples gráficos. Se utiliza con el selector libMultipleChart.
  • TableDirective: Para visualización de datos en tablas. Se utiliza con el selector libTable.

3. Configuración del Dataset

El corazón de la visualización de datos en ngx-data-visualizer es el objeto Dataset. Este objeto encapsula las dimensiones y los datos en bruto que serán procesados y visualizados.
