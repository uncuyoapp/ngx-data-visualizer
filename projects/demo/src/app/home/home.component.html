NGX-DATA-VISUALIZER

Este proyecto consiste en una aplicación Angular que incluye una librería de visualización de datos (`ngx-data-visualizer`) y un proyecto demo que muestra su implementación.

Componentes Principales:

  • NgxDataVisualizer: Una librería Angular que proporciona componentes y directivas para la visualización de datos en forma de gráficos y tablas.
  • Proyecto Demo: Un proyecto de demostración que muestra el uso de la librería NgxDataVisualizer. Incluye ejemplos prácticos y casos de uso.

Desarrollado por el Área de Políticas Públicas de la Universidad Nacional de Cuyo (APP UNCuyo). Este proyecto está licenciado bajo la Licencia MIT y se espera su uso para facilitar la visualización de datos en aplicaciones Angular.
