Esta sección proporciona una guía completa sobre el uso de la directiva libMultipleChart para la visualización de múltiples gráficos simultáneamente. Incluye configuraciones disponibles, opciones de personalización y ejemplos prácticos de implementación con separación automática por dimensiones.

Uso de la directiva libMultipleChart

La directiva libMultipleChart se utiliza para generar múltiples gráficos automáticamente basados en una dimensión específica del dataset. Cada valor único de la dimensión seleccionada genera un gráfico individual con sus propios datos filtrados. Requiere un objeto Dataset y un objeto ChartOptions como entradas.

Configuración especial para múltiples gráficos

TypeScript

Uso en el template HTML

HTML
<libMultipleChart [dataset]="dataset" [options]="chartOptions" [splitDimension]="dimension"></libMultipleChart>

2. Múltiples Gráficos de Líneas por Sector

Esta configuración presenta múltiples gráficos de líneas que muestran la evolución temporal para cada sector de gestión. Los tooltips compartidos facilitan la comparación de valores en puntos temporales específicos, mientras que las leyendas posicionadas a la derecha mantienen la identificación clara de cada serie. Este enfoque es ideal para análisis comparativos de tendencias entre diferentes categorías.

Cada gráfico circular muestra la distribución porcentual por sector de gestión, proporcionando una vista clara de las proporciones dentro de cada condición. La configuración en porcentajes y los colores vibrantes mejoran la interpretación visual de los datos categóricos.
